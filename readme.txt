== Description ==

Тази приставка ще ви помогне лесно и бързо да поставете бутоните на <a href="http://www.topbloglog.com/forum/topic.php?id=63">TopBlogLog</a> във вашите блогове.

== Installation ==

1. Изтеглете инсталационния архив на приставката, и го разархивирайте
1. Качете на вашия хостинг разархивираната папка с приставката в папката с приставките (това е <code>wp-content/plugins/</code>)
1. Активирайте приставката от страницата "Plugins" в административния панел на вашия WordPress блог.
1. Конфигурирайте приставката от страницата "Настройки" (Options) в административния панел на вашия WordPress блог.
1. Т`ва е ;)
